## 📋 What is mcp-remnawave?

mcp-remnawave is a server application designed to work with the Remnawave Panel. It manages and controls MCP (Multi-Channel Peripheral) devices connected to your system. This software lets you connect, monitor, and operate your Remnawave hardware through an easy-to-use interface.

This guide walks you through downloading and running the mcp-remnawave app on a Windows PC. You do not need any technical knowledge or programming skills to get started.

## ⚙️ Advanced Setup Tips

- **Run as Administrator**

  If you experience issues starting the server or device connection, try running the app as administrator. Right-click the app icon and select “Run as administrator.”

- **Firewall settings**

  Ensure Windows Firewall or any third-party security software allows mcp-remnawave to communicate over your network.

- **Update your devices**

  Keep your Remnawave hardware firmware up to date. Check your device’s user manual or Remnawave official site for instructions.

- **Logs and troubleshooting**

  The app creates log files for diagnostics. You can find logs inside the installation folder under `Logs`. These can help if you need support.
